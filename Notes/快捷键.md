# vscode快捷键
1. `shift` + `alt` + `down`  向下复制一行
2. `shift` + `alt` + `up`  向上复制一行
4. `ctrl` + `L`  选择整行
5. `ctrl` + `shift` + `k`  删除整行
6. `alt` + `up`/`down`  移动选中的行
7. `ctrl` + `left`/`right`  光标跳到下一个单词
8. `ctrl` + `enter`  光标在任意位置换行
9.  `ctrl` + `d` 批量更改变量（多按几次）
10. `ctrl` + `b` 隐藏文件栏
11. `shift` + `alt` + `f`  格式化代码
12. `ctrl` + `shift` + `p`  显示所有命令（菜单）
13. `ctrl` + `shift` + `f`  在整个项目文件中查找
14. `ctrl` + `p`  转到当前项目中的某个文件
15. `ctrl` + `A`  切换终端
